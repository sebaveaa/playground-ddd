# Glossary

Glosario de acceso rápido para agentes. La definición formal sigue viviendo en los `CONTEXT.md`.

- `Mission`: plantilla reusable de una experiencia.
- `Mission Stage`: etapa base de una misión.
- `Hint`: pista o solución asociada a una etapa.
- `LiveSession`: ejecución en vivo de una misión.
- `Session Team`: equipo dentro de una sesión específica.
- `Evidence Submission`: intento de resolver la etapa actual.
- `Validation Outcome`: resultado de validar una evidencia.
- `Scoreboard`: fuente de verdad del puntaje acumulado por sesión.
- `Score Entry`: registro atómico de cambio de puntaje.
- `Penalty`: descuento con motivo y momento.
- `Session Event Log`: historial auditable de eventos.
- `User`: identidad autenticable del sistema.
- `Access Token`: credencial emitida tras autenticación.
