# Agent Docs

`docs/agents/` contiene contratos operativos para agentes, no documentación detallada del producto ni una estructura futura del código.

## Objetivo

- Mantener una base común para Codex y Claude Code.
- Definir qué debe leer cada agente antes de trabajar.
- Evitar que los agentes inventen rutas o arquitectura no presentes en el repo.
